Infrared Photo Editor (All-in-One)

Steps:

  1. Upload an infrared photo.
  2. We’ll automatically swap red and blue channels.
  3. Click near a neutral area to set white balance, using the average color of a 3×3 region around the clicked pixel.
  4. Click "Save Image" to download the result if you’d like.

Tip: If you want to start over, just re-upload the image or refresh the page.
