Steps:

  1. Upload an infrared photo (JPEG or other formats).
  2. We’ll automatically swap red and blue channels.
  3. Click near a neutral area to set white balance, using the average color of a 3×3 region.
  4. Adjust Exposure, Black Point, and Saturation sliders as desired.
  5. Optional: Check “Retain Original Metadata” to keep EXIF info (only if original file was JPEG).
  6. Click “Save Image” to download the result.

Tip: If you want to start over, just re-upload the image or refresh the page.
